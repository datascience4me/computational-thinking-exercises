# Function growth

Classes 

1. O(log n), o(log n), Ω(log n) and ⍵(log n)
3. O(n), o(n), Ω(n), and ⍵(n)
4. O(n²), o(n²), Ω(n²), and ⍵(n²)
5. O(2ⁿ), o(2ⁿ), Ω(2ⁿ), ⍵(2ⁿ)

Functions:

1. f(n) = 23 × n
	1. Ω(log n) and ⍵(log n)
	2. O(n) and Ω(n)
	3. O(n²) and o(n²)
	4. O(2ⁿ) and o(2ⁿ)

2. f(n) = 42 × n² - 100 × n
	1. Ω(log n) and ⍵(log n)
	2. Ω(n) and ⍵(n)
	3. O(n²) and Ω(n²)
	4. O(2ⁿ) and o(2ⁿ)

3. f(n) = n / log(n)
	1. Ω(log n) and ⍵(log n)
	2. O(n), o(n)
	3. O(n²) and o(n²)
	4. O(2ⁿ) and o(2ⁿ)

4. f(n) = log(n) / n
	1. O(log n) and o(log n)
	2. O(n) and o(n)
	3. O(n²) and o(n²)
	4. O(2ⁿ) and o(2ⁿ)

5. f(n) = n²/log(n)

6. f(n) = log(n) + log(n)/n

	2. O(n) and o(n)
